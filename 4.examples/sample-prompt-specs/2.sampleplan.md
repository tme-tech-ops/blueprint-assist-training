## Overview
This example prompt requests a comprehensive walkthrough of building a Windows
Server VM on an ESXi host using a DAP (Dell Automation Platform) blueprint,
without actually implementing the blueprint.

Without building the blueprint, walk me through an example of how to build a
Windows Server VM running on an ESXi host using a DAP blueprint.
